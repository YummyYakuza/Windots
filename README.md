🪟 Windows 11 Dotfiles

Personal configuration files for my Windows 11 setup, focused on customization, productivity, and making the OS slightly less annoying.

---

📌 Overview

This repository contains my dotfiles and configuration setup for Windows 11.
It includes system customization tools, launcher configs, terminal settings, and development environment tweaks.

---


⚙️ Tools Used

Click on any tool to view installation steps and documentation:

- "YASB" (https://github.com/amnweb/yasb) – Status bar customization
- "Komorebi" (https://github.com/LGUG2Z/komorebi) – Tiling window manager for Windows
- "Windows Terminal" (https://github.com/microsoft/terminal) – Terminal emulator
- "Windhawk" (https://github.com/ramensoftware/windhawk) – Windows customization mods
- "Flow Launcher" (https://github.com/Flow-Launcher/Flow.Launcher) – App launcher / productivity tool
- "VS Code" (https://github.com/microsoft/vscode) – Code editor

---

🚀 Setup

1. Clone the repository

git clone https://github.com/your-username/windows-11-dotfiles.git
cd windows-11-dotfiles

---

2. Install Required Tools

Go through each tool listed above and follow their official installation guides.

(Yes, you actually have to click the links. I know, exhausting.)

---

3. Apply Configurations

- Copy or symlink config files to their respective locations
- Restart applications after applying configs
- Ensure required fonts/themes are installed

---

🧠 Notes

- Designed specifically for Windows 11
- Some configs may depend on installed fonts, themes, or external tools
- Expect minor adjustments depending on your system
